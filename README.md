# 📊 SQL & Python Sales Data Analysis

A data analysis project built using **MySQL** and **Python** to analyze sales data, generate business insights, and visualize trends using charts.

## 📌 Project Overview

This project demonstrates how SQL and Python can be combined to analyze business data efficiently. The project performs data extraction, cleaning, analysis, and visualization to help identify sales trends and support data-driven decision-making.

## 🎯 Objectives

- Analyze sales performance
- Extract meaningful insights using SQL queries
- Process data using Python
- Create visualizations for better understanding
- Practice real-world data analysis techniques

## 🛠️ Technologies Used

- Python
- MySQL
- Pandas
- Matplotlib
- Jupyter Notebook
